# INTERFACE-GRAFICA-001 — AURA/ARGUS

## Objetivo
Implementar a primeira camada gráfica real do produto AURA/ARGUS, substituindo a aparência de placeholder por uma identidade visual própria, futurista e coerente com a arquitetura definida.

## Implementado
- Landing page cinematográfica em 16:9 responsiva.
- Identidade visual dark HUD.
- Avatares visuais de AURA e ARGUS em CSS.
- Narrativa mitológica e conceitual dos nomes.
- Cockpit de dashboard com estado real do projeto.
- Chat visual preparado para integração com IA.
- Login e cadastro com estética do produto.
- Sidebar e header atualizados.
- Componentes reutilizáveis adicionados.

## Componentes novos
- `components/ui/avatar-orb.tsx`
- `components/ui/status-pill.tsx`
- `components/ui/section-title.tsx`
- `components/ui/metric-row.tsx`

## Observação
Esta entrega é visual e estrutural. Ela não implementa ainda autenticação real, IA real, memória real ou ações reais. Essas camadas devem vir depois.
